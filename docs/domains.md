## Abstract 
a lot of the times, code is fundamentally unreadable because we got lost within the layers, of abstraction, of security, of air handling, and we often lose the core idea of the program. this is because programming as a language, is ambiguous when red, but when executed by a computer, it is absolutely non-ambiguous.
